# Editor_Texto

Editor de Texto criado utilizando a Windows Forms e a linguagem C#, desenvolvido conhecimentos nos componentes do Windows Forms, lógica e sintaxe da linguagem. Todas as aplicações funcionando:

-Criar novo Arquivo
-Abrir
-Salvar
-Copiar
-Colar
-Imprimir
-Sair
-Negrito
-Italico
-Sublinhado
-Centralizar
-Alinhar a esquerda
-Alihar a direita
